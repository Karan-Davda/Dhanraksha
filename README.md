# Dhanraksha

Personal Finance Manager (PWA) — **React + TypeScript + Nhost + Three.js** (per `Personal_Finance_Manager_Complete_Specification.docx`).

## Getting started

- Frontend lives in `web/`.
- Spec lives at `Personal_Finance_Manager_Complete_Specification.docx`.

### Run web app

```bash
cd web
npm install
cp env.example .env.local
# edit .env.local with your Nhost project values
npm run dev
```
